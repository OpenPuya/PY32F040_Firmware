================================================================================
                                样例使用说明
                             Sample Description
================================================================================
功能描述：
此样例演示了时钟输出功能，可输出PLL波形（32MHz）。

Function descriptions:
This sample demonstrates the clock output function, which can output the PLL 
waveform (32MHz).
================================================================================
测试环境：
测试用板：PY32F040_STK
MDK版本： 5.28
IAR版本： 9.20
GCC 版本：GNU Arm Embedded Toolchain 10.3-2021.10

Test environment:
Test board: PY32F040_STK
MDK Version: 5.28
IAR Version: 9.20
GCC Version: GNU Arm Embedded Toolchain 10.3-2021.10
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. 可由示波器观察到PA08输出PLL波形（32MHz）。

Example execution steps:
1. Compile and download the program to the MCU and run it.
2. The PLL waveform (32MHz) can be observed on PA08 using an oscilloscope.

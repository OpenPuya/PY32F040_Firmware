================================================================================
                                样例使用说明
                             Sample Description
================================================================================
功能描述：
此样例演示了通过软件方式将进入STOP模式改为复位芯片。

Function descriptions:
This example demonstrates how to change the mode of entering STOP to resetting 
the chip through software
================================================================================
测试环境：
测试用板：PY32F403_STK
MDK版本： 5.28
IAR版本： 9.20

Test environment:
Test board: PY32F403_STK
MDK Version: 5.28
IAR Version: 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. 按下按键，程序会设置进入STOP模式改为复位芯片，设置成功，会点亮LED。

Example execution steps:
1. Compile and download the program to MCU and run it;
2. Press the button,The program will set to enter STOP mode and reset the chip. 
If the setting is successful, the LED will light up.

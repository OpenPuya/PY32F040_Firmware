================================================================================
                                样例使用说明
                             Sample Description
================================================================================
功能描述：
此样例演示了USB打印机的枚举功能。

Function descriptions:
This sample demonstrates the enumeration function of the USB printer.

================================================================================
测试环境：
测试用板：STK
MDK版本：5.28

Test environment:
Test board: STK
MDK Version: 5.28

================================================================================
使用步骤：
1. 下载并运行程序；
2. 通过USB连接线，一端连接STK板，另一端连接PC电脑；
3. PC端显示连接打印机；

Example execution steps:
1. Download and run the program;
2. Connect the STK board to the PC computer using the USB cable;
3. The PC will display the connection of the printer;

================================================================================
注意事项：
1. 此样例只实现了USB打印机枚举过程，因为打印机的数据传输过程跟具体的设备有关，
   不宜做成demo。

Notes:
1. This sample only demonstrates the enumeration process of the USB printer. 
   The data transmission process of the printer is device-specific and not 
   suitable for demonstration purposes.

================================================================================
